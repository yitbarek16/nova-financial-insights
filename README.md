# Nova Financial Insights Analysis

## Overview

This project analyzes the relationship between financial news sentiment and stock price movements for the Nova Financial Solutions. It combines exploratory data analysis, technical stock analysis, and sentiment-based correlation to assess whether news sentiment has a measurable impact on market behavior.

The work was completed as part of the 10 Academy – Artificial Intelligence Mastery Program.

## Objectives

- Perform exploratory analysis on financial news data
- Analyze stock price behavior using technical indicators
- Measure correlation between news sentiment and daily stock returns
- Build a reproducible and well-tested analytics workflow

## Project Components

- Task 1 – Data & Infrastructure

  - Git-based workflow with CI/CD
  - Exploratory data analysis of financial news

- Task 2 – Quantitative Stock Analysis

  - Technical indicators (SMA, EMA, RSI, MACD, Bollinger Bands, ATR)
  - Financial metrics (VWAP, OBV, volatility, returns)
  - Visualization of price and volume patterns

- Task 3 – News–Market Correlation

  - Sentiment analysis using VADER
  - Alignment of news and stock price data
  - Pearson correlation between sentiment and returns

## Key Findings

- Technical indicators effectively capture trends, momentum, and volatility
- Volume-based metrics provide early market signals
- News sentiment shows varying correlation strength across stocks
- Data alignment and coverage significantly impact sentiment–price analysis

## Tech Stack

- Python
- Pandas, NumPy
- Matplotlib, Seaborn
- NLTK (VADER), spaCy
- TA-Lib, yfinance
- Scikit-learn
- Jupyter Notebook
